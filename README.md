# Python Deep Learning Notes

This repository contains Python learning code for PyTorch, NLP, RNNs, attention, Transformer, and transfer learning examples.

## Structure

- `DL_4/`: PyTorch and neural network examples.
- `NLP/`: Text preprocessing, sequence models, attention, Transformer, and transfer learning examples.

Large generated artifacts, IDE files, caches, and saved model files are intentionally ignored.
